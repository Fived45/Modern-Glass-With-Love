# Fived's YASB Theme
Literally the Modern Glass theme but ~it's transparent~ (yeah not anymore, black background), has less widgets (they're more useful I think? idk you can edit it however you want), and has a cool looking Cava widget integrated already.

# How to install:
Replace the existing style.css and config.yaml files in C:\Users\USERNAME\ .config\yasb with these ones, and you're good to go :D
